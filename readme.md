# Narci - 币安高频交易数据处理与回测系统

Narci 是一个专为加密货币量化交易者设计的全流程开发框架。它专注于解决币安（Binance）市场微观数据的获取、清洗、订单簿重建（OrderBook Reconstruction）及策略回测难题，为高频交易（HFT）和做市策略研究提供精准的数据基础设施。

## 🛠 功能模块

本系统采用模块化设计，核心组件如下：

### 1. 数据获取与管理
* **历史数据下载器 (`download.py`)**：
    * 自动化下载币安 L1 (aggTrades) 和 L2 (depth) 每日数据。
    * 自动解压 ZIP 并转换为高性能 parquet 格式。
    * 支持多线程并发下载，大幅提升数据准备效率。
* **实时数据录制器 (`l2_recoder.py`)**：
    * 基于 WebSocket 实时捕获深度增量（Diff）和逐笔成交。
    * 自动同步 REST API 快照，确保数据流序列的绝对完整。
    * 具备断线重连与定期落盘机制，适合长期运行的数据采集。

### 2. 数据处理与重构
* **订单簿重构引擎 (`l2_reconstruct.py`)**：
    * **L3 级重建**：基于原始增量数据还原完整的订单簿状态。
    * **特征工程**：计算盘口不平衡度 (Imbalance)、买卖价差 (Spread) 及微观流动性指标。
    * 提供采样快照功能，将高频 Tick 数据降采样为固定时间窗口（如 100ms）的特征数据集。
* **数据校验工具 (`validator.py`)**：
    * 校验本地文件 MD5 与官方校验和是否一致。
    * 业务逻辑检查：检测 ID 断层、异常价格（<=0）及时间戳单位错误。

### 3. 策略研究与回测
* **事件驱动回测引擎 (`backtest.py`)**：
    * 内置 `SimulatedBroker` 模拟交易所撮合逻辑。
    * 支持自定义手续费率与资金管理。
    * 生成详细的交易报告（收益率、平均成交价等）。
* **策略基类 (`strategy.py`)**：
    * 提供易于扩展的策略模板。
    * 包含双均线/突破策略的参考实现 (`MyTrendStrategy`)。

### 4. 可视化交互
* **数据分析看板 (`dashboard.py`)**：
    * 基于 Streamlit 开发的 Web 界面。
    * 支持 K 线图 (OHLC) 与成交量叠加展示。
    * 独有的“多空主动成交分布”分析，直观展示市场买卖压力。

## 🚀 快速开始

1.  **安装依赖**
    ```bash
    pip install pandas numpy streamlit plotly aiohttp websockets pyarrow requests
    ```

2.  **获取数据**
    ```bash
    # 下载历史数据
    python download.py
    
    # 或启动实时录制
    python l2_recoder.py
    ```

3.  **运行回测**
    ```bash
    python backtest.py
    ```

4.  **启动看板**
    ```bash
    streamlit run dashboard.py
    ```

## 🔮 愿景 (Vision)

Narci 致力于打造一个**轻量级、高精度、低门槛**的量化研究基础设施。

我们的核心目标是：
1.  **数据平权**：消除个人开发者处理交易所 L2/L3 级别海量数据的技术壁垒，让微观结构数据不再是机构的专利。
2.  **无缝衔接**：构建统一的数据标准，打通从“历史数据清洗”到“策略回测”再到“实盘交易”的全链路流程。
3.  **回归本质**：让量化交易者从繁琐的数据工程（Data Engineering）中解放出来，专注于策略逻辑本身的数学与金融价值。
